# SuperApp Full Package

This package includes:

- Laravel backend skeleton with APIs for jobs, actions, workers, DB download
- React frontend skeleton (Dashboard component)
- Database schema (MySQL) with users, jobs, actions, platform accounts
- Payment & referral split scaffolding
- Messaging/chat placeholder (WebSocket-ready)
- Deployment-ready folder structure

## Steps
1. Configure `.env` in `backend/` for DB connection
2. Import `db/schema.sql` into MySQL
3. Serve Laravel backend (`php artisan serve`) in `backend/`
4. Integrate frontend into your SPA and build React app
